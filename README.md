# Stereo Vision depth estimation using Python

Using Python and open CV in order to estimate the depth of an object or a point using the intersection of 2 images

# How to use:

1- install numpy, tqdm and OpenCV
2- run stereo_calibration_driver.py if you want to use a different dataset(adjust the parameters!)
3- change left and right image that you want to estimate thier depth(Depth_estimator.py)
4- run Depth_estimator.py
